Gmail2Drive

Save selected Gmail emails as PDFs — with their attachments — straight to your Google Drive, organised into folders.

Get started Privacy

What it does

One folder per conversation

Each selected conversation is saved into its own folder, named after the email subject.

Whole thread in one PDF

Every message in a conversation is combined into a single, tidy PDF.

Attachments included

All attachments are saved alongside the PDF, with duplicates from quoted replies removed.

Your Drive, your choice

Pick any destination folder, including Shared Drives. Inline logos and signatures are embedded in the PDF.

Getting started

  1. Install Gmail2Drive (via the private install link shared with you).
  2. Open Gmail and select one or more conversations.
  3. Click Save to Drive in the toolbar.
  4. Approve the one-time Google permission prompt, then choose a destination folder.

Gmail2Drive is currently distributed privately. To use it, your Google account must be added to the access list — contact the developer if you need access.

Privacy first

Everything runs in your browser and talks only to Google's own Gmail and Drive APIs using your authorization. No external servers, no tracking, and nothing is collected. Read the full privacy policy.
